Module: Grid - Introduction

The grid is the basic module to provide boxes for all of your modules. It is a global module, which means it is not scoped and can (also should) be used everywhere on the site.

The grid-wrapper class provides the white box to hold the module as well as the padding and margin.

The grid class actually holdes the module and provides a flexbox-box as well as a grid-system for the modules.

Module: Grid - full width

The element class indicates an element within the grid-box. The grid-system can be divided in 3 elements (width of one element = x) that can also be combined.

The x3 class indicates an element, which combines the width of all 3 elements (width of one element = x). The x3 is the short form of x*3.

Module: Grid - 2:1 / 1:2

The element class indicates an element within the grid-box. The grid-system can be divided in 3 elements (width of one element = x) that can also be combined.

The x2 class indicates an element, which combines the width of 2 elements (width of one element = x). The x2 is the short form of x*2.
The x1 class indicates an element, with the width of 1 element.

Both elements can be placed in their natural order and will take their given space.

Module: 1:1:1

The element class indicates an element within the grid-box. The grid-system can be divided in 3 elements (width of one element = x) that can also be combined.

The x1 class indicates an element, with the width of 1 element. You can also use the x1-fix class, if you need an element that does not change the width of one grid element, but is fixed.

All elements can be placed in their natural order and will take their given space. You can also wrap the elements together as seen in the next module.

1 - I am fixed.
2 - I will vary my width.
3 - I will vary my width.
					
<article class="grid">
	<div class="element x1-fix"> 1 - I am fixed.</div>
	<div class="element x1"> 2 - I will vary my width.</div>
	<div class="element x1"> 3 - I will vary my width.</div>
</article>

Module: Hero Tiles

Row of 3 tiles. Creates a "call-to-action" stopper for the main page. This module is scoped and needs the use of the ID.

The hero-tile id indicated the hero tile module.

After that we need to add the tiles class which gives us a grid-system for full-width elements. tiles-x3 continues the width pattern as see in our global grid system, but as a special use case as it does not provide any paddings / margins.

A tile-item class indicates a tile item. The colors are alternating. Also in responsive mode. The items have a min-width, but will expand the height of the row in relation to the content.

Module: Bio

This module is a 1:1:1 module, with a fixed first element to display a picture. The second element is for description or bio of a person, including name, where this person is from etc. The last of the elements is for contact information.
